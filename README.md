# Features

* Using different databases for development (sqlite3) and production (postgres)
* Followed the tutorial
* Tests made with [`RSpec`](https://relishapp.com/rspec/)

* `ruby` v2.5.5
* `rails` v5.2.3+
* RVM
* `vscode` with _Ruby_ and _Rubocop_ extensions
* Linux
